This app uses express session and passport.js to authenticate users.

## Address

## Technologies
Project is created with:
+ express v4.17.3
+ node.js v16.14.2
+ npm v8.5.0
+ PostgreSQL v14.2
+ VSCode v1.65.2

Allows clients to register and log in.
